### 1.2 Опрацювання команд Lab 9 - Archiving and Compression
| Назва команди | Її призначення та функціональність |
|---|---|
| cd | Команда **cd** використовується для переходу між директоріями у файловій системі. У даному випадку вона переміщує користувача до домашнього каталогу. |
| mkdir mybackups | Команда **mkdir** використовується для створення нової директорії. У цьому випадку створюється папка **mybackups** у домашньому каталозі користувача для збереження архівних копій файлів. |
| tar -cvf mybackups/udev.tar /etc/udev | Команда **tar** використовується для об’єднання декількох файлів або директорій у один архівний файл. У цьому випадку вміст директорії **/etc/udev** архівується у файл **udev.tar**, який зберігається у директорії **mybackups**. Параметр **-c** означає створити новий архів. Параметр **-v** (verbose) показує процес виконання команди. Параметр **-f** використовується для вказання назви архівного файлу. |
| ls mybackups | Команда **ls** використовується для перегляду вмісту директорії. У цьому випадку вона показує список файлів, що знаходяться у папці **mybackups**. |
| tar -tvf mybackups/udev.tar | Команда **tar** використовується для перегляду вмісту архівного файлу без його розпакування. Параметр **-t** відображає список файлів у архіві. Параметр **-v** показує детальну інформацію про файли. Параметр **-f** вказує назву архіву. |
| tar -zcvf mybackups/udev.tar.gz /etc/udev | Команда **tar** використовується для створення архіву та одночасного стиснення за допомогою **gzip**. Параметр **-z** означає використання алгоритму стиснення gzip. Параметр **-c** створює новий архів. Параметр **-v** відображає процес виконання. Параметр **-f** задає ім’я архівного файлу. |
| ls -lh mybackups | Команда **ls** використовується для перегляду вмісту папки **mybackups**. Параметр **-l** відображає детальну інформацію про файли, а параметр **-h** показує розмір файлів у зручному для читання форматі. |
| cd mybackups | Команда **cd** змінює поточну директорію на **mybackups**, де зберігаються архіви. |
| ls | Команда **ls** використовується для перегляду файлів і директорій у поточній папці. |
| tar -xvf udev.tar.gz | Команда **tar** використовується для розпакування архіву. Параметр **-x** означає витягнути файли з архіву. Параметр **-v** показує процес розпакування. Параметр **-f** задає ім’я архівного файлу. |
| ls etc | Команда **ls** відображає вміст директорії **etc**, яка була створена після розпакування архіву. |
| ls etc/udev | Команда **ls** використовується для перегляду файлів у директорії **etc/udev**. |
| ls etc/udev/rules.d | Команда **ls** показує файли у директорії **rules.d**, яка знаходиться у папці **udev**. |
| tar -rvf udev.tar /etc/hosts | Команда **tar** використовується для додавання нового файлу до вже існуючого архіву. У цьому випадку файл **/etc/hosts** додається до архіву **udev.tar**. Параметр **-r** означає додати файл до архіву. |
| cp /usr/share/dict/words . | Команда **cp** використовується для копіювання файлів. У цьому випадку файл **words** копіюється з директорії **/usr/share/dict** у поточну директорію. Символ **.** означає поточну папку. |
| ls -l words | Команда **ls** використовується для перегляду інформації про файл **words**. Параметр **-l** показує детальну інформацію про файл: права доступу, власника, розмір та дату створення. |
| gzip words | Команда **gzip** використовується для стиснення файлів. У цьому випадку файл **words** стискається і створюється файл **words.gz**. При цьому оригінальний файл замінюється стисненим. |
| ls -l words.gz | Команда **ls** використовується для перегляду інформації про файл **words.gz**, включаючи його розмір, права доступу та дату створення. |
| gunzip words.gz | Команда **gunzip** використовується для розпакування файлів, стиснених за допомогою **gzip**. У цьому випадку файл **words.gz** розпаковується і відновлюється файл **words**. |
| bzip2 words | Команда **bzip2** використовується для стиснення файлів за допомогою алгоритму **bzip2**. У цьому випадку створюється файл **words.bz2**, а оригінальний файл замінюється стисненим. |
| ls -l words.bz2 | Команда **ls** використовується для перегляду інформації про файл **words.bz2**, включаючи його розмір, права доступу та дату створення. |
| bunzip2 words.bz2 | Команда **bunzip2** використовується для розпакування файлів, стиснених за допомогою **bzip2**. У цьому випадку файл **words.bz2** розпаковується і відновлюється файл **words**. |
| xz words | Команда **xz** використовується для стиснення файлів за допомогою алгоритму **xz**. У цьому випадку файл **words** стискається і створюється файл **words.xz**. |
| ls -l words.xz | Команда **ls** використовується для перегляду інформації про файл **words.xz**, включаючи його розмір, права доступу та дату створення. |
| unxz words.xz | Команда **unxz** використовується для розпакування файлів, стиснених за допомогою **xz**. У цьому випадку файл **words.xz** розпаковується і відновлюється файл **words**. |
| zip words.zip words | Команда **zip** використовується для створення архіву у форматі **zip**. У цьому випадку створюється архів **words.zip**, який містить файл **words**. Оригінальний файл при цьому залишається без змін. |
| ls -l words.zip | Команда **ls** використовується для перегляду інформації про файл **words.zip**, включаючи його розмір, права доступу та дату створення. |
| zip -r udev.zip /etc/udev | Команда **zip** використовується для створення архіву **udev.zip**, який містить директорію **/etc/udev** та всі її файли і піддиректорії. Параметр **-r** означає рекурсивне архівування. |
| ls -l udev.zip | Команда **ls** використовується для перегляду інформації про архівний файл **udev.zip**. |
| unzip -l udev.zip | Команда **unzip** використовується для перегляду вмісту архіву **zip** без його розпакування. Параметр **-l** відображає список файлів у архіві. |
| rm -r etc | Команда **rm** використовується для видалення файлів і директорій. Параметр **-r** означає рекурсивне видалення директорії **etc** разом з усім її вмістом. |
| unzip udev.zip | Команда **unzip** використовується для розпакування архіву **udev.zip** у поточну директорію. |
